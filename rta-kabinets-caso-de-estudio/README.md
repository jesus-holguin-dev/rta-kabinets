# RTA Kabinets Solutions - Case Study

Caso de estudio de una plataforma web desarrollada para una carpintería, enfocada en presentación de servicios, administración de productos y generación de estimaciones para clientes.

## Tecnologías utilizadas

- React
- Node.js
- Express
- MySQL
- JavaScript
- HTML
- CSS
- Dokploy
- Hostinger

## Funcionalidades principales

- Página pública para presentar servicios y productos
- Galería de trabajos
- Formulario de contacto
- Panel administrativo
- Administración de productos
- Manejo de variantes de productos
- Generación de estimaciones para clientes

## Evidencia del proyecto

Este repositorio puede incluir capturas de pantalla, descripción funcional y resumen técnico del proyecto.

## Nota importante

Por tratarse de un proyecto utilizado por un cliente, el código fuente completo no se publica. Este repositorio funciona como caso de estudio para mostrar el alcance, tecnologías y funcionalidades implementadas sin exponer código privado o información sensible.
