# Resumen del proyecto

RTA Kabinets Solutions es una plataforma web creada para apoyar la presencia digital de una carpintería y mejorar la presentación de servicios, productos y estimaciones para clientes.

## Problema

El negocio necesitaba una forma más clara de mostrar productos, organizar información y generar estimaciones para clientes interesados en trabajos de carpintería y remodelación.

## Solución

Se desarrolló una aplicación web con frontend, backend, base de datos y despliegue en hosting, incluyendo una sección administrativa para gestionar contenido y apoyar la generación de estimaciones.

## Módulos principales

- Página pública
- Productos
- Galería
- Contacto
- Panel administrativo
- Gestión de productos
- Variantes de productos
- Generador de estimaciones

## Tecnologías

- React
- Node.js
- Express
- MySQL
- JavaScript
- HTML
- CSS
- Dokploy
- Hostinger
